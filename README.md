# Cricket Chairman

A Vercel-ready Next.js cricket club chairman simulation.

You start as a tiny village cricket club and try to climb into the County Championship. If you win Division One, England Selector / Ashes mode unlocks.

## Features

- 10-season save
- Week-by-week chairman decisions
- 7-division cricket pyramid
- Promotion and relegation
- 8-team leagues
- Match simulation
- Squad management
- Transfers
- Contract renewals
- Manager and staff hiring
- Youth pulls
- Facilities
- Ticket/merch/food pricing
- Fan satisfaction
- Media pressure
- Inbox decision events
- LocalStorage autosave
- Export/import save JSON
- Ashes selector reward mode

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in terminal.

## Build

```bash
npm run build
```

## Deploy to Vercel

Push this repo to GitHub, then import it in Vercel.

## Notes

This is a full playable MVP. It is deliberately fictional and does not use real player databases.
